Welcome to RE-EMISSION's documentation

Version: |version|

**Re-Emission** is a Python library and a command line interface (CLI) tool for estimating greenhouse gas (GHG) emissions from reservoirs. It calculates net life-cycle biogenic emissions as well as emission profiles over time for each of the three greenhouse gases: $CO_2$, $CH_4$ and $N_2O$.
